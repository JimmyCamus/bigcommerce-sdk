## Getting Started

### Prerequisites

- PNPM

  ```sh
  npm install -g pnpm
  ```

- or NPM

  ```sh
  npm install npm@latest -g
  ```
### Clone the repo

```sh
git clone https://github.com/JimmyCamus/bigcommerce-sdk.git
```

### Install dependencies

- PNPM

  ```sh
  pnpm i
  ```

- or NPM

  ```sh
  npm i
  ```

### Build the sdk package

- PNPM

  ```sh
  pnpm build-sdk
  ```

- or NPM

  ```sh
  npm build-sdk
  ```

### Run the project

- PNPM

  ```sh
  pnpm dev
  ```

- or NPM

  ```sh
  npm dev
  ```
